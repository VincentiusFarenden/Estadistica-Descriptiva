# Estadística Descriptiva

Conjunto de ejercicios en Jupyter Notebook sobre estadística descriptiva con Python.

## Contenido

- Medidas de tendencia central (media, mediana, moda)
- Medidas de dispersión (varianza, desviación estándar)
- Análisis de distribuciones
- Visualización de datos estadísticos
- Tablas de frecuencia
- Probabilidad y azar
- Modelos de regresión lineal
- Distribución normal de probabilidad
- Medidas de asociación
- Coeficiente de determinación

## Instalación

Clona el repositorio:
```bash
git clone https://github.com/VincentiusFarenden/Estadistica-Descriptiva.git
cd Estadistica-Descriptiva
```

Instala las dependencias:
```bash
pip install numpy pandas matplotlib seaborn scipy jupyter
```

## Uso

Inicia Jupyter Notebook:
```bash
jupyter notebook
```

Abre los notebooks (.ipynb) y ejecuta las celdas para ver los ejercicios y resultados.

## Librerías utilizadas

- NumPy
- Pandas
- Matplotlib
- Seaborn
- SciPy
